# Yashar Sanaei - [Wasmer.io](https://wasmer.io) hiring task

This repository is a small React starter cloned from `https://github.com/wasmerio/edge-react-starter.git` for the
Wasmer\.io hiring process. It contains a simple React application to show calendar like events in a timeline container.

---

_**The idea behind this architecture is to keep it minimal and simple with having this in mind that it might become a 
complex product in the future. 
So, Expansion was in mind but the simplicity was the priority.**_

--- 

## Features

- React frontend scaffold
- Example WebAssembly integration (from the original starter)
- Minimal setup to run locally and build for production

## Requirements

- Node\.js (LTS)
- npm

## Quick start

1. Install dependencies: `npm install`
2. Run in development mode:`npm run dev`
3. Build for production:`npm run build`

## Project structure

- `src/` \- React source files
- `public/` \- static assets
- `package.json` \- scripts and dependencies
